## Installation


### Prerequisites
- python 3

### Steps
1. Clone the repository: `git clone https://github.com/Zod55/Youtube-Download.git`
2. Navigate to the project directory: `cd Youtube-download`
3. Install dependencies: `pip3 install moviepy pytube`
